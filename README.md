# Cosmetics E-Commerce Platform

A full-stack e-commerce web application for selling cosmetic products.

## Features

- Product listing and detail pages
- Shopping cart functionality
- Backend integration
- Responsive design
- CRUD operations for products

## Tech Stack

Frontend:
- React / JavaScript / Vite/ Bootstrap

Backend:
- Node.js / Express / Spring Boot

## Project Purpose

This project was developed to practice full-stack development, API integration, and state management in a real-world e-commerce scenario.
